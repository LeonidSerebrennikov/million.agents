В первой и третьей части ответы на вопросы и диаграмма


Во второй части практическое задание, для запуска перейти в папку "часть 2(практическая)" и ввести ```docker-compose up --build```. Будет собран образ на докере со всеми необходимыми зависимостями. 

По работе самого приложения: все эндпоинты представлены на localhost:8000/docs, для создания проекта нужно сначала создать пользователя, т.к. есть проверка на существование owner'a. Для этого нужно отправить пустой post на /users. Затем post-запросом на /projects создать проект с id этого пользователя, который будет возвращен в ответе, и любым, но не пустым названием. И уже после этого создать задачу, где указать название, project_id и author_id. Остальные поля заполнить по желанию. В получении списка задач работает опциональная фильтрация и пагинация.
